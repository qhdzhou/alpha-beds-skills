# Alpha Beds Hotel Booking

Alpha Beds gives AI agents a complete hotel booking API — search 700,000+ hotels worldwide with natural language, browse rooms, verify real-time rates, and book with crypto payment, all through a single REST API.

## Available Skills

| Skill | Payment | Install |
|-------|---------|---------|
| `alpha-beds-hotel-fluxa` | USDC via FluxA | `npx skills add -s alpha-beds-hotel-fluxa -y -g qhdzhou/alpha-beds-skills` |

## Install

```
npx skills add -s alpha-beds-hotel-fluxa -y -g qhdzhou/alpha-beds-skills
```

After installation, follow the **Booking Flow** in the skill to search and book hotels.
